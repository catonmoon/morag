# Естественный интеллект: устройство мозга и механизмы познания

Естественный интеллект — способность биологических систем воспринимать информацию, обучаться на опыте, решать задачи и адаптироваться к новым условиям. В отличие от искусственных нейросетей, мозг эволюционировал сотни миллионов лет и реализует многие функции принципиально иначе.

---

## Нейрон: базовая вычислительная единица

### Строение нейрона

Нейрон — специализированная клетка, передающая электрохимические сигналы:

```
           Дендриты (вход)
               ╱╲ ╱╲ ╱╲
              ╱  ╲╱  ╲╱
             [  Soma  ]  ← тело клетки (суммирование входов)
              ╲
               ╲═══════════════════╗  Аксон (длинный отросток)
                                   ║
                              [Миелин] [Миелин] [Миелин]
                                   ║
                          [Аксонный терминал]
                                   ↓
                          Синаптическая щель
                                   ↓
                     Дендриты следующего нейрона
```

**Основные части:**
- **Дендриты** — древовидные отростки, принимающие сигналы от других нейронов
- **Сома (тело)** — суммирует входящие сигналы
- **Аксон** — длинный отросток, передающий сигнал (до 1 метра у человека)
- **Миелиновая оболочка** — изоляция аксона, ускоряет передачу сигнала в 10–100 раз
- **Синаптические терминали** — выходы к следующим нейронам

### Потенциал действия

Нейрон работает по принципу «всё или ничего»: если суммарный входящий сигнал превышает порог (~−55 мВ), генерируется потенциал действия — электрический импульс:

```
Напряжение
    │         ╭──╮
+40 │         │  │  (пик потенциала действия)
    │         │  │
  0 │─────────╯  │
    │             ╲
−55 │ · · · · · · ·╲ · ·  ← порог возбуждения
    │               ╲
−70 │────────────────╲────  ← потенциал покоя
    │                 ╰──── гиперполяризация
    └──────────────────────────── время (мс)
```

Частота потенциалов действия кодирует интенсивность сигнала: тихий звук — 10 Гц, громкий — 200–300 Гц.

### Сравнение нейрона и искусственного нейрона

| Характеристика | Биологический нейрон | Искусственный нейрон |
|---|---|---|
| Входы | 1 000–100 000 дендритов | Фиксированное число весов |
| Выход | Спайк (двоичный) или частота | Вещественное число |
| Обучение | Синаптическая пластичность (часы–дни) | Градиентный спуск (секунды) |
| Энергопотребление | ~20 Вт (весь мозг) | 300–700 Вт (GPU для инференса LLM) |
| Тип вычислений | Аналоговый + временно́й | Цифровой, синхронный |
| Скорость | 1–1000 Гц | Миллиарды операций в секунду |

## Синапсы и обучение

### Синаптическая передача

Синапс — место контакта двух нейронов. Сигнал передаётся химически через нейромедиаторы:

```
Пресинаптический нейрон
         │
    [Везикулы с нейромедиатором]
         │
         ↓  (потенциал действия → выброс)
    ════════════════  ← синаптическая щель (20–40 нм)
         ↓
    [Рецепторы постсинаптической мембраны]
         │
Постсинаптический нейрон
```

Основные нейромедиаторы и их роль:

| Нейромедиатор | Тип | Функции |
|---|---|---|
| Глутамат | Возбуждающий | Обучение, память, восприятие |
| ГАМК | Тормозной | Торможение, снижение тревоги |
| Дофамин | Модулятор | Система вознаграждения, мотивация |
| Серотонин | Модулятор | Настроение, сон, аппетит |
| Ацетилхолин | Возбуждающий | Внимание, память, мышечное управление |
| Норадреналин | Модулятор | Внимание, реакция на стресс |

### Правило Хебба и синаптическая пластичность

«Нейроны, которые активируются вместе, связываются вместе» (Хебб, 1949):

```
Если нейрон A регулярно участвует в возбуждении нейрона B,
то синаптическая связь A→B усиливается.
```

Молекулярный механизм — **LTP (Long-Term Potentiation)**:
1. Глутамат связывается с NMDA-рецепторами
2. При совпадении пресинаптической и постсинаптической активности — Ca²⁺ входит в клетку
3. Активируются протеинкиназы → больше AMPA-рецепторов встраивается в мембрану
4. Синапс становится более «чувствительным» на часы, дни, годы

Это и есть клеточная основа памяти и обучения.

## Структура мозга

### Доли коры головного мозга

```
           Лобная доля          Теменная доля
           (планирование,      (пространство,
            речь, решения)      тело, числа)
              ╭────────────────────╮
             ╱                    ╲
            ╱   КОРА БОЛЬШИХ       ╲
           │    ПОЛУШАРИЙ           │
            ╲                      ╱
             ╲                    ╱
              ╰─────────────────╯
           Затылочная доля       Височная доля
           (зрение)             (слух, память,
                                 распознавание)
```

| Зона | Функции | Повреждение |
|---|---|---|
| Префронтальная кора | Планирование, торможение импульсов, рабочая память | Потеря самоконтроля, расстройства личности |
| Моторная кора | Управление движениями | Паралич соответствующих частей тела |
| Сенсорная кора | Обработка тактильных ощущений | Потеря чувствительности |
| Зона Брока | Производство речи | Брока-афазия: понимает, но не говорит |
| Зона Вернике | Понимание речи | Вернике-афазия: говорит, но бессвязно |
| Зрительная кора | Обработка визуальной информации | Слепота (при двустороннем поражении) |

### Гиппокамп: центр памяти

Гиппокамп — структура в медиальной височной доле — критически важен для формирования новых воспоминаний:

```
Новый опыт
     │
     ▼
[Гиппокамп]  ← кодирование (минуты–часы)
     │
     ▼
[Консолидация] ← повторная активация во сне
     │
     ▼
[Кора больших полушарий] ← долгосрочное хранение
```

**Случай H.M. (1953)**: пациент Генри Молисон после удаления гиппокампа полностью потерял способность формировать новые декларативные воспоминания, но сохранил старые и мог обучаться новым моторным навыкам. Это разделило память на:

- **Декларативная** (явная): факты и события — зависит от гиппокампа
- **Процедурная** (имплицитная): навыки и привычки — не зависит от гиппокампа

### Мозжечок: автоматизация движений

Мозжечок содержит **~50% всех нейронов мозга** при занимаемом объёме ~10%. Функции:

- Координация движений и равновесие
- Точное управление мышечным тонусом
- Обучение двигательным последовательностям (езда на велосипеде, игра на инструменте)

Интересно: мозжечок, по-видимому, реализует форму предиктивного кодирования — постоянно сравнивает ожидаемый и реальный результат движения.

### Базальные ганглии и система вознаграждения

Базальные ганглии участвуют в выборе действий и обучении с подкреплением:

```
Кора (намерение действия)
         │
         ▼
[Стриатум] ← дофаминовый сигнал от VTA/substantia nigra
         │
         ▼
[Паллидум/Субталамус] — фильтрация
         │
         ▼
[Таламус]
         │
         ▼
Кора (исполнение)
```

Дофамин кодирует **ошибку предсказания вознаграждения** (prediction error) — именно этот механизм лёг в основу алгоритма TD-обучения в RL:
- Получил больше, чем ожидал → всплеск дофамина
- Получил меньше, чем ожидал → провал дофамина
- Получил ровно столько, сколько ожидал → нет изменений

## Виды памяти

### Классификация

```
Память
├── Декларативная (явная, сознательная)
│   ├── Эпизодическая  ← личные события («вчера я был на встрече»)
│   └── Семантическая  ← факты и знания («Париж — столица Франции»)
│
└── Недекларативная (неявная, автоматическая)
    ├── Процедурная    ← двигательные навыки (езда на велосипеде)
    ├── Прайминг       ← неосознанное влияние предыдущего опыта
    ├── Условный рефлекс ← классическое обусловливание (Павлов)
    └── Привыкание     ← снижение реакции на повторяющийся стимул
```

### Рабочая память

Рабочая память — «оперативная память» мозга. Характеристики:

- **Объём**: 7 ± 2 элемента (Миллер, 1956); уточнение — 4 ± 1 (Коуэн, 2001)
- **Длительность**: 15–30 секунд без повторения
- **Зависимость от**: префронтальная кора + теменная кора

```python
# Аналогия с программированием:
# Рабочая память ≈ переменные в текущей функции (стек вызовов)
# Долгосрочная память ≈ база данных / файловая система
# Процедурная память ≈ скомпилированный код (не нужно «думать» при выполнении)
```

Нарушения рабочей памяти характерны для СДВГ, шизофрении, нормального старения.

## Обучение и нейропластичность

### Виды пластичности

**Структурная пластичность** — физические изменения в мозге:
- Рост новых синапсов (синаптогенез)
- Рост новых нейронов (нейрогенез) — доказан для гиппокампа
- Удаление неиспользуемых связей (синаптический прунинг)

**Функциональная пластичность** — изменение силы существующих связей (LTP/LTD).

### Критические периоды

Мозг особенно пластичен в определённые периоды развития:

| Функция | Критический период | Последствия пропуска |
|---|---|---|
| Бинокулярное зрение | 0–5 лет | Амблиопия («ленивый глаз») |
| Родной язык (фонематика) | 0–12 лет | Акцент при изучении позже |
| Второй язык (грамматика) | 0–15 лет | Труднее усвоить нативную грамматику |
| Музыкальный слух | 0–9 лет | Совершенный слух редок без раннего обучения |

### Правило 10 000 часов

Исследования К. Андерса Эрикссона показали: экспертиза в любой области требует ~10 000 часов **целенаправленной практики** (deliberate practice):

- Работа на пределе возможностей
- Немедленная обратная связь
- Фокус на слабых сторонах

Простое повторение без этих условий не даёт экспертного роста (пример: большинство людей водят машину 10+ лет, не становясь профессиональными гонщиками).

## Восприятие и предиктивное кодирование

### Мозг как машина предсказаний

Современная нейронаука рассматривает мозг не как пассивный «приёмник» информации, а как активную **предсказательную машину**:

```
Мозг постоянно генерирует предсказания о входных данных
         │
         ▼
Реальные сенсорные данные поступают
         │
         ▼
Вычисляется ошибка предсказания
         │
    ┌────┴────┐
    ▼         ▼
Обновить    Обновить
предсказание  веса модели
(если ошибка  (если ошибка
  мала)       велика)
```

Это объясняет многие иллюзии — они возникают, когда предсказание мозга «побеждает» реальные данные.

### Иерархия обработки зрения

Зрительная система — лучший изученный пример иерархической обработки:

```
Сетчатка → Края и контрасты
    ↓
V1 (первичная зрительная кора) → Ориентация линий
    ↓
V2/V4 → Форма, цвет
    ↓
IT (нижняя височная кора) → Объекты, лица
    ↓
Префронтальная кора → Смысл, контекст
```

Каждый уровень кодирует всё более абстрактные признаки — именно эту идею позаимствовали глубокие нейросети (CNN).

## Сознание и интеллект

### Теории сознания

| Теория | Авторы | Суть |
|---|---|---|
| Глобальное рабочее пространство | Баарс, Деэн | Сознание — «трансляция» информации в глобальную сеть мозга |
| Интегрированная информация (IIT) | Тонони | Сознание = мера интегрированной информации (Φ) |
| Предиктивная обработка | Фристон | Сознание — иерархическая предиктивная модель мира |
| Высший порядок (HOT) | Розенталь | Сознательный опыт = мысль о мысли |
| Квантовое сознание | Пенроуз, Хамерофф | Квантовые процессы в микротрубочках (спорно) |

Ни одна теория не является консенсусной. «Трудная проблема сознания» (Чалмерс, 1995) — почему физические процессы порождают субъективный опыт — остаётся открытой.

### g-фактор и структура интеллекта

Психометрические исследования выявили **общий фактор интеллекта (g)** — статистическая корреляция между различными когнитивными тестами:

```
         g (общий интеллект)
        ╱ │ ╲ │ ╲
       ╱  │  ╲│  ╲
     Gf  Gc  Gsm  Gv   ...
      │   │   │    │
    тесты тесты тесты тесты
```

- **Gf** (fluid intelligence) — рассуждение, решение новых задач; снижается с возрастом
- **Gc** (crystallized intelligence) — накопленные знания; растёт до 60–70 лет
- **Gsm** — рабочая память
- **Gv** — пространственное мышление

Дискуссия о природе g продолжается: одни считают его реальной биологической характеристикой, другие — статистическим артефактом.

## Сравнение: мозг и LLM

| Характеристика | Мозг человека | GPT-4 / Claude |
|---|---|---|
| Параметры | ~100 трлн синапсов | ~1 трлн параметров |
| Энергопотребление | ~20 Вт | ~300–500 Вт (GPU) |
| Скорость обучения | Годы | Недели (предобучение) |
| Few-shot обучение | Мгновенно (1–3 примера) | Хорошо, но не всегда |
| Обобщение | Очень гибкое | Ограничено обучающим распределением |
| Воплощение | Да (тело, сенсоры) | Нет |
| Эмоции | Да (влияют на решения) | Нет (симуляция) |
| Долгосрочная память | Неограниченная | Только в пределах контекста |
| Забывание | Постепенное, адаптивное | Резкое (за пределами контекста) |
| Многозадачность | Последовательная (иллюзия параллельности) | Один запрос |
| Устойчивость к шуму | Высокая | Зависит от задачи |
| Творчество | Да | Рекомбинация обученного |
| Понимание причинности | Интуитивное | Ограниченное |

### Чего мозг не умеет (а LLM умеет)

- Мгновенный поиск по миллиардам документов
- Точные арифметические вычисления над большими числами
- Воспроизведение дословных длинных текстов
- Параллельная обработка одного запроса на тысячах копий
- Моментальное «клонирование» знаний в новую копию модели

### Чего LLM не умеет (а мозг умеет)

- Непрерывное обучение без забывания (catastrophic forgetting)
- Обучение на единственном примере (one-shot) в физическом мире
- Каузальное понимание (понимание причин, а не корреляций)
- Реальное планирование в условиях неопределённости
- Сознательный опыт и квалиа
- Адаптация к физической среде через тело

## Нарушения и болезни

### Нейродегенеративные заболевания

| Болезнь | Поражаемая система | Симптомы | Механизм |
|---|---|---|---|
| Болезнь Альцгеймера | Гиппокамп → кора | Потеря памяти, деменция | Амилоидные бляшки, тау-белок |
| Болезнь Паркинсона | Чёрная субстанция | Тремор, скованность | Гибель дофаминовых нейронов |
| БАС | Моторные нейроны | Паралич, атрофия мышц | Дегенерация моторного нейрона |
| Болезнь Хантингтона | Стриатум | Непроизвольные движения | Мутация гена HTT (наследственная) |

### Психические расстройства и нейронауки

- **Депрессия** — снижение активности префронтальной коры, уменьшение объёма гиппокампа; дефицит серотонина и норадреналина
- **Шизофрения** — нарушения дофаминовой системы, аномальная связность сетей; ошибки предиктивного кодирования (галлюцинации как «некорректируемые предсказания»)
- **СДВГ** — нарушения префронтальной регуляции, дефицит дофамина и норадреналина в цепях внимания
- **ПТСР** — гиперактивность миндалины, нарушение консолидации памяти в гиппокампе

## Сон и консолидация памяти

Сон — не пассивное состояние, а активная фаза обработки информации:

```
Бодрствование: сбор новых воспоминаний (гиппокамп)
     │
     ▼
NREM-сон (медленные волны):
  - Реактивация воспоминаний дня
  - «Перенос» из гиппокампа в кору
  - Sleep spindles ← связывают гиппокамп и кору
     │
     ▼
REM-сон (быстрые движения глаз):
  - Ассоциативная обработка
  - Эмоциональная разрядка воспоминаний
  - Творческие связи между удалёнными концептами
     │
     ▼
Утро: улучшенное запоминание, инсайты
```

Исследование Стикголда (2000): задача на ротацию фигур решалась на 33% лучше после сна, чем после равного времени бодрствования — независимо от времени суток.

«Ночной сон» перед экзаменом эффективнее ночного зубрежа — это не метафора, а нейробиологический факт.

## Эволюционная перспектива

### Трёхуровневый мозг (модель Маклина)

Упрощённая, но полезная метафора эволюционных слоёв:

```
Неокортекс (приматы)
 ├─ Рассуждение, язык, планирование
 ├─ Абстрактное мышление
 └─ Самосознание

Лимбическая система (млекопитающие)
 ├─ Эмоции (страх, любовь, злость)
 ├─ Социальные связи
 └─ Долгосрочная память

Рептильный мозг (ствол + мозжечок)
 ├─ Базовые инстинкты (еда, размножение, безопасность)
 ├─ Регуляция тела (дыхание, сердцебиение)
 └─ Автоматические реакции
```

Конфликты между уровнями объясняют многие «иррациональные» аспекты человеческого поведения: страх публичных выступлений (лимбическая система) при понимании его безопасности (кора), тяга к нездоровой еде и т.д.

### Размер мозга и интеллект

Мозг человека: ~1300–1400 г, ~86 миллиардов нейронов, ~100 трлн синапсов.

Но объём мозга коррелирует с интеллектом внутри вида лишь слабо (r ≈ 0.3). Важнее:
- **Нейронная плотность** (особенно в коре)
- **Связность** между зонами (белое вещество)
- **Метаболическая эффективность**

Слон имеет мозг в 3× больше человеческого, но явно уступает по когнитивным способностям — потому что большая часть его нейронов находится в мозжечке для управления хоботом.
